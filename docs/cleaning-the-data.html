Chapter 2 Cleaning the data

Although there is plenty to clean in this data, the most critical pieces currently are ensuring that the species codes are correct (four characters, all capitalized, no extra spaces, codes are correct for species) and to make sure that any instances where distance and species code data were switched are switched to the correct columns.
